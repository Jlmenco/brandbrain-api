# Market Agent - Cadastro de Fontes e Concorrentes

## Fontes recomendadas (MVP)
1) RSS/Notícias setoriais (portais, associações)
2) Google Trends (termos por nicho)
3) Relatórios públicos (PDFs e blogs oficiais)
4) YouTube/Podcast (títulos e descrições públicas) - opcional

## Padrão de cadastro
- Fonte macro do Grupo: cost_center_id = null
- Fonte específica de marca: cost_center_id = {ccId}

## Concorrentes/referências
- Cadastrar por marca (ccId)
- Campos mínimos:
  - name
  - website_url
  - social_handles (opcional)
  - notas (ex.: “referência em conteúdo educativo”)
