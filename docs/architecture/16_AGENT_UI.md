# Marketing Agent - UI/UX

## Tela: Agent (por Cost Center)
- Chat (lado esquerdo)
- Painel "Ações propostas" (lado direito)
  - Lista de drafts a criar
  - Canais alvo (checkbox)
  - Botões:
    - "Criar drafts"
    - "Enviar para revisão"
    - "Gerar calendário da semana"
    - "Adaptar do Master"

## Componentes
- Quick actions:
  - Criar influencer
  - Gerar 10 ideias
  - Montar calendário semanal
  - Gerar posts da semana
  - Adaptar macro do Master

## UX de segurança
- Sempre mostrar "Preview" antes de executar criação em lote.
- Exigir confirmação para ações de alto volume (ex.: 30 drafts).
